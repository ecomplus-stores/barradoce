# Descrição do produto

Jogo de assadeiras super prático para fazer o famoso bolo de camadas (Naked Cake).
 
São 2 assadeiras baixas, para fazer cada uma das camadas do bolo. Cada uma tem 1,9cm de espessura.

As assadeiras são aço de ótima qualidade, com reveistimento antiaderente e proporcionam um perfeito cozimento da massa.

* Material: aço antiaderente
* Kit com 2 assadeiras
* Medida de cada assadeira: 22,8 x 33,0 x 1,9cm
